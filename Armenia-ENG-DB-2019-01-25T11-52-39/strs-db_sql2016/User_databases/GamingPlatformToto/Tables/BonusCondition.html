Ամեն մի դինամիկ փոփոխվող բոնուս ունի իր պայմանները, որի ընթացքում և կատարվում են բոնուսի պայմանների  փոփոխություններն: Ըստ բոնուսի նախորոք որոշված կանոնների կատարվում են այդ փոփոխություններն: Դինամիկ փոփոխվող բոնուսի պայմանների մասին ինֆորմացիա պարունակվում է այդ աղյուսակում: Բոնուսի պայմանների նկարագրությունը և բիզնես պրամաբանությունը հստակ չէ, և երբեմն ծրագրավորողներն հարդ կոդ են արել, այս կամ այն տեսակի բոնուսներ կայքում աշխատաց ելու համար:  Each dynamic bonus has his conditions. During this condition the changes were made. These changes will be processed according to the business logic of the changes. This table contains information about these bonus conditions.
