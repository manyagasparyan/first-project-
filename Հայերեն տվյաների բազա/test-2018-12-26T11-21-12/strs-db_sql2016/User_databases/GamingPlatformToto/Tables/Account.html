[dbo].[Account]
MS_Description
This table contains information about players, wallets types. The player or user wallets can be unused, used and bonus types. Unused type of wallet is that wallet that the withdraw have been made for the user, unused wallet is that wallet for what the withdrawal have been done, but the amount is yet in the process, and the bonuses wallet is the separate wallet used only for bonuses.
