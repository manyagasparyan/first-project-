[dbo].[AccessObject]
MS_Description
 Այս աղյուսակտ ստեղծված է օգտագործողների համար , որպեսզի
տանաջատվի օգտագործողների հասանելիություններն և տեսանելիությունը որոշակի ինֆորմացիայի
շրջանակում: Ստեղծված է օգտագործողի համար, որպեսզի հնարավոր լինի  պարտնյորներին վերաբերվող ինֆորմացիան տարանջատել
և հասանելի դարձնել միայն որոշակի օգտագործողների խմբին պլատֆորմայի ադմին պանելում:
Այս աղյուսակը կարող է պարունակել դեպոզիտ, withdraw , կամ պարտնորի  ID  թվերը:Սրանք այն օբեկտներ ի մի մասն  են,  որոնք ծրագրավորողներն  օգտագործում են այս աղյուսակի շրջանակում:  This table created for the user in order to differentiate users visibility in the circle of some information. This information created for use in order to skip e one partner information from other partner on the platform environment, objects on this table contain deposit, withdraw, and partner ID numbers.
